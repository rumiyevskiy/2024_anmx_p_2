Section 1

Тут використовується адаптивний розмір шрифту для заголовка. Представлено дві версії. перша - адаптивна, друга - статична (не змінна). При зміні розміру екрану перша буде мінятись, друга ні.

font-size: calc(24px + 16 * ((100vw - 320px) / (1280 - 320)));
	тут:
		24px - це розмір шрифту по макету при мобільній версії сторінки, або, якщо немає макету на мобільну версію,
                то це мінімальний розмір шрифту при мобільному варіанті сторінки який ми сами обираємо в браузері використовуючи
                інструменти розробника і там змінюючи розмір шрифта до оптимального розміру. 
		16 - це різниця між розмірами шрифту по макету між десктопною та мобільною версіями.
		100vw - ширина viewport, тобто ширина екрана.
		320px - мінімальний розмір екрана мобільної версії
		1280 - це ширина сторінки по макету
		320 - мінімальний розмір екрана мобільної версії

для scss :

$maxWidth: 1200;

@mixin adaptiv-font($pcSize, $mobSize) {
 $addSize: $pcSize - $mobSize;
 $maxWidth: $maxWidth - 320;
 font-size: calc(#{$mobSize + px} + #{$addSize} * ((100vw - 320px) / #{$maxWidth}));
}

.page__section__title {
    @include adaptiv-font(40, 24);
    margin: 0 0 10px 0;
}
	тут:
		$maxWidth - розмір сторінки по макету.
		$pcSize - розмір шрифту для пк версії.
		$mobSize - розмір шрифту для мобільної версії.
		@include adaptive-font(40, 24) - це підключення адаптивної властивості для  css файлу для елементу з класом .page__section__title через 
                міксини (scss) з вказівкою у дужках розмірів: 40 - для пк версії, 24 - для мобільної версії
